## Featured Projects

### [InternScrapes](https://github.com/Yug-Vashisth/Internscrapes)
Automated internship scraper that monitors major tech companies and banks, stores postings in SQLite, filters duplicates, and sends scheduled email digests.

**Highlights**
- Tracks 8+ major companies
- Stores 1000+ unique postings
- Sends grouped HTML job digests
- Cuts manual internship searching dramatically

### [New York Giants Gameday Predictor](https://github.com/Yug-Vashisth/Giants_ML_Predictor)
Machine learning project that predicts New York Giants game outcomes using historical performance data, betting lines, weather, and custom football features.

**Highlights**
- Feature engineering for QB trends, injuries, and defensive metrics
- Multiple classification models tested
- 70%+ validation accuracy
- Automated weekly scraping and preprocessing pipeline
